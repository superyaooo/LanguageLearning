Form template example

This is an attempt to make a generic standardized form stylesheet with different column, row options. It is based on Tyler's beautiful forms.

  • For double column rows, elements are "inline-block", instead of table cells
  • The space between two columns won't change in case of different label lengths
  • element width is by percentage%
